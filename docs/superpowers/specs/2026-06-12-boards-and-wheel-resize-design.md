# 便籤板 + 滾輪縮放 設計

日期:2026-06-12

## A. 滾輪縮放

- 游標懸停在元素上滾滾輪即縮放,不需先選中
- 文字:fontSize ×1.1 / ÷1.1,上下限 8–400px(與拖曳 handle 相同)
- 圖片:width ×1.1 / ÷1.1,下限 24px
- 板子:懸停在板內某條 → 只調那一條;懸停在標題列或板底 → 整板等比縮放(標題 + 所有項目 + 寬度)
- 滾動停止後 debounce 300ms 存快照

## B. 便籤板

- 建立:輸入框打 `#標題` → 生成空板,不加新熱鍵
- 結構:標題列(可拖曳移動整組、雙擊改名、顏色可換)+ 直列清單(自動排版,板子自動長高)
- 寬度固定(預設 280px,最小 160px),右下 handle 或滾輪可調;高度隨內容
- 進出:散落便籤拖到板上顯示插入線,放開收進;板內拖動重排序;拖出範圍變回自由便籤
- 板內項目可雙擊編輯、用工具列改樣式
- 刪除:選中板按 Del 或垃圾桶 → 整組刪(板 + 所有項目)
- 不做:收合、巢狀板、勾選完成

## C. 資料

快照新增 kind:

```json
{ "kind": "board", "x": 0, "y": 0, "width": "280px", "title": "今天要做",
  "color": "#ffd400", "titleFontSize": "16px",
  "items": [{ "kind": "text", "text": "買牛奶", "fontSize": "20px", "style": {} }] }
```

## D. 檔案

- `board.js`(新):板子建立/序列化/進出轉換/插入線/整板縮放
- `overlay.js`:wheel 監聽、mousedown 項目拖曳、dblclick 標題與項目、`#` 前綴、快照整合
- `toolbar.js`:板專用工具列(換色 + 垃圾桶)
- `overlay.html`:板子 CSS、載入 board.js
